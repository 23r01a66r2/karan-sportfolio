# karan-sportfolio
Vasudevana Karan — Portfolio Website
📌 Overview

This is a modern, responsive personal portfolio website built to showcase my skills, projects, experience, and achievements as a Computer Science student.

The website features a sleek UI, smooth animations, and a well-structured layout designed to highlight both technical and professional capabilities.

🚀 Features
🎨 Modern UI with custom cursor and animations
📱 Fully responsive design
⚡ Smooth scrolling and section-based navigation
💼 Projects showcase with tech stacks
🧠 Skills and technologies categorized
🏆 Achievements & certifications section
📞 Contact section with email, phone, and LinkedIn
🛠️ Technologies Used
Frontend:
HTML5
CSS3 (Advanced styling, animations, grid, flexbox)
JavaScript (Vanilla JS)
Design:
Google Fonts (Syne, DM Mono, Fraunces)
Custom UI/UX styling with gradients and effects
